# **App Name**: Depot Dispatch Display

## Core Features:

- Real-time Departure Board: Display Carrier, Destination, Via, Trailer Number, Collection Time, Bay Door, Seal Number, Schedule Number, and Status.
- Dynamic Status Coloring: Automatically color-code entries based on status: Green (Departed), Yellow (Loading), Blue (Waiting), Red (Cancelled), Orange (Delayed).
- Admin Interface for Schedule Management: Allow authorized personnel to modify the departure schedule, including editing details and statuses, without coding.
- Current Date and Time Display: Show the current date and time prominently on the display.
- Departure Notifications: Send a message when a given truck changes status to departed.
- Route Optimization Tool: AI powered tool that will offer improved truck routes.

## Style Guidelines:

- Primary color: Deep blue (#1A237E), reminiscent of logistics and trust.
- Background color: Very light grey (#F5F5F5), a muted tone for comfortable viewing in a waiting area.
- Accent color: Muted purple (#9FA8DA), providing enough differentiation from the primary color, indicating a call for attention.
- Body and headline font: 'Inter', a sans-serif for clarity on digital displays.
- Use a tabular layout for clear organization of departure information.
- Incorporate simple icons to represent each Carrier.
- Implement subtle transitions when updating the status of a departure.
